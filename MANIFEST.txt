FILE DESCRIPTIONS
=================

BMP280_Mesh_Sensor.ino (Basic)
  - 4.2 KB
  - Simple sensor node firmware
  - Reads BMP280 sensor every 30 seconds
  - Transmits via mesh network
  - No external dependencies beyond headers
  - Good for learning

BMP280_Mesh_Advanced.ino (Advanced)
  - 6.8 KB
  - Full mesh networking features
  - Neighbor discovery & tracking
  - Packet forwarding with hop limit
  - Statistics tracking
  - CRC16 validation
  - Advanced logging

sensor_data.h
  - 500 bytes
  - SensorReading structure (21 bytes)
  - MeshPacket wrapper structure
  - Data type definitions

mesh_utils.h
  - 624 bytes
  - CRC16 calculation (X.25 polynomial)
  - Hex encoding/decoding
  - Utility functions for mesh operations

platformio.ini
  - 278 bytes

  - WEMOS D1 Mini board settings
  - Library dependencies
  - Build flags

QUICK START STEPS
=================

1. Extract COMPLETE.zip

2. Navigate to Basic_Version or Advanced_Version

3. Wire hardware:
   - BMP280: SDA→D2, SCL→D1
   - LoRa: CS→D8, RST→D4, DIO0→D3

4. Install PlatformIO:
   pip install platformio

5. Build & upload:
   pio run -e d1_mini --target upload

6. Monitor:
   pio device monitor -b 115200

HARDWARE REQUIREMENTS
====================

- ESP8266 WEMOS D1 Mini
- BMP280 sensor module (I2C)
- LoRa radio module (SPI)
- 3.3V power supply
- USB cable for programming

TECHNICAL SPECS
===============

Firmware Size:
  Basic: 5.79 KB
  Advanced: 9.43 KB

Sensor Data Packet: 26 bytes total
  - Header: 5 bytes
  - Payload: 21 bytes
  - CRC16: 2 bytes included

LoRa Radio:
  - Frequency: 915.0 MHz (US)
  - Bandwidth: 125 kHz
  - Spreading Factor: 9
  - Coding Rate: 7
  - TX Power: 17 dBm

Power Consumption:
  - Idle: ~5 mA
  - Sensor reading: ~15 mA
  - LoRa TX: ~100 mA peak
  - Typical average: 7-10 mA

Battery Life (2000 mAh):
  - ~273 hours (11.4 days)
  - With 30-second sensor interval

CONFIGURATION
=============

Edit in sketch before upload:

Radio Parameters:
  LORA_FREQ = 915.0 MHz
  LORA_BW = 125.0 kHz
  LORA_SF = 9
  LORA_CR = 7
  TX_POWER = 17 dBm

Sensor Parameters:
  SENSOR_INTERVAL = 30 seconds
  BMP280_I2C_ADDR = 0x76

Network Parameters:
  NODE_ID = 0x0001
  HOP_LIMIT = 5
  NODE_NAME = "Sensor_01"

SUPPORT & TROUBLESHOOTING
=========================

BMP280 Issues:
  - Check I2C address (0x76 or 0x77)
  - Verify SCL/SDA wiring
  - Test with multimeter (3.3V supply)

LoRa Issues:
  - Verify SPI connections
  - Check antenna attachment
  - Match gateway frequency
  - Try external antenna for range

Memory Issues:
  - Use basic version for ESP8266
  - Check heap usage in serial output
  - Reduce MAX_NEIGHBORS in advanced version

Building:
  - Ensure PlatformIO is installed
  - Check board selection: d1_mini
  - Verify USB port
  - Use 115200 baud for upload
